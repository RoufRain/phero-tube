Discuss the scope of var,let and const

Var: The scop of 'var' variable is functional scope. It can be updated and re-declared into the scope.

Let: The scope of 'let' variable is block scope. It can be updated but cannot be re-declared into the scope.

Const: The scope of 'const variable is block scope. It cannot ve updated or re-declared into the scope.'

Tell us the use cases of null and undefined

Undefined: Undefined means a variable has been declared but has yet not been assigned a value.

Null: Null is an assignment value. It can be assigned to a variable as a representation of no value.

What do you mean by REST API?

A REST API (RESTful API) is an application programming interface (API) that conforms to the constraints of REST architectural style and allows for interaction with RESTful web services. REST stands for representational state transfer.
